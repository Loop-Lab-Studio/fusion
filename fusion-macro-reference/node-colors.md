# Fusion Macro Reference: Node Colors

A list of Fusion node colors used in `.setting` files.


## Location

```lua
{
	Tools = ordered() {
		Tool_Name = Custom {
			CtrlWZoom = false,
			Inputs = {},
			ViewInfo = OperatorInfo {},

			Colors = { 
        TileColor = { 
          R = 0.92156862745098, 
          G = 0.431372549019608, 
          B = 0 
        }, 
      },

			UserControls = ordered() {}
		},
		Tool_NameLUTIn1 = LUTBezier {},
		Tool_NameLUTIn2 = LUTBezier {},
		Tool_NameLUTIn3 = LUTBezier {},
		Tool_NameLUTIn4 = LUTBezier {}
	},
	ActiveTool = "Tool_Name"
}
```
